IoT-Based Battery Safety Monitoring System for Electric Motorcycle

CDE4301 Innovation and Design Capstone — AY2024/2025

Abstract

This report presents the design and implementation of a prototype IoT-based battery safety monitoring system for a high-performance electric motorcycle developed by an industry partner. The company's existing hardware lacked battery safety monitoring. This project extends it by adding temperature monitoring using the DS18B20, voltage monitoring using an ADC resistor divider circuit, and motion data logging using the MPU6050 IMU. All sensor data is uploaded every second via the AIR780EX 4G module to the OneNET cloud platform and viewable on a mobile app. When any value exceeds a configured threshold, the platform sends an email alert. A local OLED display provides real-time on-device feedback. Four out of five objectives were fully met. The main contribution is system integration. No existing commercial product currently combines battery monitoring, motion data logging, GPS reporting, and 4G connectivity in one device designed for electric motorcycles.
